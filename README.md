# CMPM120-D1-Cinematic

This cinematic demonstrates the intro for my game: Wonhyo

---------------------------------------------------------

The cinematic includes four scenes, including the references scene. 

The cinematic indludes an audio file for arrow deflection and another for background music. 

The cinematic uses 5+ cited images that display monk drawings, arrows, and a papyrus background to allude a sense of lost history. 

The cinematic uses Wonhyo as the title text. This was used with https://cooltext.com/

The cinematic uses two rectangles behind the title text to create the allusion of an outlined background slider. 

The cinematic fades to black upon transitioning from the title scene to the references scene. 

The cinematic includes multi line text here: "A really long time ago... /n there lived a monk..."

The cinematic uses a variety of moving objects, the most notable being the arrow deflection action.
